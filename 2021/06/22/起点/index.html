# 初衷

其实自己早就有一个建站的想法，只是一直无从起笔。大学的两年时间过去的如此之快，自己好像并没有落下些什么。因此，我决定写下些东西，来应对这飞逝的时光。

上了大学以后，工科的环境很压抑，有时候会感觉自己很麻木，很多事情并不能让自己很愉悦，也有很多事情也并不击我至沮丧，我的情绪波动渐渐没有那么大了... 我并不认为喜怒不形于色是一件好事，想让自己情绪化一些，所以，本站就这样诞生了。

# 起点

前两天和好朋友聚餐，提了我的想法，但不知道从哪里开始写起。从高中写起，事情纷纷杂杂，不聚；想了想，从毕业写起，正正好。

我有一个八人的小团体，我们一起走过了高中三年时间，我们每个人的经历不尽相同。三年的时间，我们像兄弟至亲一样支撑着彼此。行至今日，我们依然保持着紧密的联系。关于我们每个人，性格各异，也因此碰撞出很多火花。关于我的回忆，就从这里写起。

所谓起点，那年毕业，是我的起笔之处，也是我们故事的开始。
